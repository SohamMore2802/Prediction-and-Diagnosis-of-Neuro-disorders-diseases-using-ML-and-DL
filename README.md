# Prediction-and-Diagnosis-of-Neuro-disorders-diseases-using-ML-and-DL

This project focuses on building and validating machine learning models to predict and diagnose neurodegenerative and neurocognitive disorders such as Alzheimer’s disease, Parkinson’s disease, Huntington’s disease and ALS
The objective is to support clinicians with data-driven decision tools that improve early detection, diagnostic accuracy, and patient outcomes. The system integrates multimodal data including neuroimaging (MRI, PET), genetic markers, clinical assessments, cognitive scores, and electronic health records
The workflow includes data preprocessing, normalization, feature extraction, and handling missing values. Imaging features such as hippocampal volume and cortical thickness are extracted, along with genetic and clinical indicators.
Models implemented include classical machine learning algorithms (SVM, Random Forest), deep learning architectures (CNNs, RNNs), and Graph Neural Networks for connectivity analysis.
Model performance is evaluated using accuracy, precision, recall, AUC, sensitivity, specificity, and k-fold cross-validation.
Interpretability methods like SHAP and LIME are used to enhance clinical trust.
The final system is designed for deployment as a clinical decision support tool via API and web interface, enabling scalable and practical healthcare integration
